### 坐标计算规则

- **起始 X 坐标**: `280`
- **格子宽度**: `173`
- **Y 坐标**: `1384`
- **计算公式**: `x = 280 + (index - 1) * 173`

| 格子序号 | X 坐标 | 计算方式 |
|---------|--------|----------|
| 1 | 280 | 280 |
| 2 | 453 | 280 + 173 |
| 3 | 626 | 280 + 173 × 2 |
| 4 | 799 | 280 + 173 × 3 |
| ... | ... | ... |
| n | 280 + (n-1) × 173 | 280 + 173 × (n-1) |

## 窗口激活 vs 未激活的坐标差异

> **注意**: 窗口未激活时，Window/Client 坐标为负数（相对于桌面），而激活后为正数。建议使用 **Screen 坐标** 或 **相对于窗口左上角的偏移量**。

|----------|---|---|

| Client | -840 | 990 |
